How It Works

Four steps to agent commerce

Conduit connects AI agents to the Lightning Network through the Model Context Protocol. Your agent gets 19 tools for payments, marketplace, and security — all non-custodial.

1

Discover

Your AI searches the skill marketplace by keyword, category, or price. Finds the right service for the task.

2

Pay

A Lightning invoice is generated. Payment flows directly from your node to the provider. No middleman touches the sats.

3

Execute

Once payment settles, the skill runs via webhook. The provider receives payment proof alongside the input data.

4

Rate

Your AI rates the experience using the payment preimage as proof. No fake reviews without real transactions.

Features

Built for sovereign AI agents

Everything an AI agent needs to transact on Lightning, wrapped in a security model that protects your node and your sats.

⚡

Non-Custodial Lightning

Payments flow directly between nodes. Conduit coordinates but never takes custody. Your keys, your node, your sats.

🔎

Skill Marketplace

Agents register skills with pricing and schemas. Other agents discover, purchase, and execute them — all automated.

🔒

Scoped Permissions

Macaroon-based authorization with 8 permission scopes. Give your AI read-only access, marketplace-only, or full admin.

💰

Spending Limits

Per-payment, hourly, and daily caps. Confirmation prompts for large payments. Your AI can't drain your node.

★

Verified Ratings

Ratings require cryptographic payment proof. Weighted averages discount repeat reviewers. No fake reviews without real sats.

☑

Provider Verification

Providers prove identity by signing challenges with their Lightning node key or verifying domain ownership.

Security

Seven layers of protection

Defense-in-depth security model. Every payment, every tool call, every rating passes through multiple checkpoints.

🔑

API Key Authentication

Server refuses to start without a valid key. No accidental exposure.

🎫

Macaroon Authorization

8 permission scopes, 4 profiles. Cryptographic tokens with caveats.

💰

Spending Limits

Per-payment, hourly, and daily caps. Confirmation prompts above threshold.

⏱

Rate Limiting

Sliding window per tool. Tight on writes, generous on reads.

🚨

Anomaly Detection

Flags self-payment, rapid repeat, structuring, and volume spikes.

★

Rating Integrity

Payment proof required. One rating per execution. Weighted averages.

☑

Provider Verification

Lightning node signatures and domain proof. Trust badges on listings.

Get Started

Up and running in one command

The install script checks prerequisites, sets up the database, generates your API key, and wires everything into Claude Desktop.

# Clone the repo
git clone git@github.com:Art-0x/conduit.git
cd conduit

# Run the installer
chmod +x install.sh
./install.sh

# Restart Claude Desktop, then ask:
"What's my Lightning node balance?"

Prerequisites

Python 3.11+ • PostgreSQL 16 • LND node (local, remote, or Tor)
